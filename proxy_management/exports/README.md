# 代理匯出資料夾

此資料夾包含可供使用者下載的代理相關檔案，分為兩個主要類別：

## 📁 working-proxies/ - 已驗證的工作代理
包含經過驗證可用的代理伺服器，品質較高且穩定。

### 檔案說明：
- `working_proxies.csv` - 主要的工作代理清單，包含回應時間和地理位置資訊
- `best_proxies.json` - 最佳代理的 JSON 格式清單
- `best_proxies.txt` - 最佳代理的文字格式清單

### 使用情境：
- 需要高品質代理進行重要任務
- 要求穩定連線和快速回應的應用
- 生產環境的代理需求

## 📁 proxy-lists/ - 原始代理列表
包含從 Proxifly 獲取的原始代理列表，未經驗證。

### 檔案說明：
- `all.txt` - 所有類型的代理（HTTP、SOCKS4、SOCKS5）
- `http.txt` - HTTP 代理專用列表
- `socks4.txt` - SOCKS4 代理專用列表  
- `socks5.txt` - SOCKS5 代理專用列表
- `us_proxies.txt` - 美國地區的代理列表

### 使用情境：
- 需要大量代理進行初步篩選
- 特定協議類型的代理需求
- 地理定位要求的代理選擇
- 自行驗證和篩選的需求

## ⚠️ 重要提醒

1. **品質差異**：working-proxies/ 中的檔案已經過驗證，品質遠高於 proxy-lists/ 中的原始列表
2. **更新頻率**：建議定期重新下載以獲取最新代理
3. **驗證建議**：使用 proxy-lists/ 中的代理前，建議先進行驗證
4. **數量差異**：working-proxies/ 通常數量較少但品質高，proxy-lists/ 數量多但需篩選

## 🔧 檔案格式

- **CSV 格式**：`IP:PORT,類型,回應時間(ms),國家,代理URL`
- **JSON 格式**：結構化代理資訊，包含完整中繼資料
- **TXT 格式**：簡單的 `IP:PORT` 列表，每行一個代理
